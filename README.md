该仓库为FPGA的训练题集，使用的教程为小梅哥的AC620视频教程，开发环境为Quartus II 18.1版本

# 开发平台

开发平台选用凌智电子的WinnerI开发板，采用Altera飓风四系列EP4CE10F17C8



 # 项目结构

altera_practice  
│  README.md  
│  
├─docs	用于存放相关资料、学习笔记等  
└─SRC	 用于存放实验代码  
     └─test	实验名称  
          ├─ip	存放IP核  
          ├─prj	存放Quartus项目  
          └─src	存放实验Verilog、dbf等
